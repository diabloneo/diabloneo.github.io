AWS SDK for Go 2.0 Developer Preview

https://aws.amazon.com/cn/blogs/developer/aws-sdk-for-go-2-0-developer-preview/

变化：

1. 去掉了session，只保留config，使用更简单。
2. 支持API Request方法，允许对每个请求进行相关设置后再发送。
3. API资源支持强类型。
4. 直接支持slice和map类型了，不用再转成指针形式。

目前2.0版本处于开发者preview状态。1.0版本将会继续维护很长一段时间，不用担心。
